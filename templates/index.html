Proxy Master

One script to set it all!

Download

set_proxy.sh

Instructions

To set proxy:

1. Open the downloads directory in the terminal
2. Execute sudo bash set_proxy.sh -h [PROXY HOST] -p [PROXY PORT]
3. Enter proxy host (172.16.2.30 for KGP) and proxy port (8080 for KGP)

To unset proxy:

1. Open the downloads directory in the terminal
2. Execute sudo bash set_proxy.sh -u

Known Issues

Some Users might need to change Settings->Network->Proxy to Manual from Off.
